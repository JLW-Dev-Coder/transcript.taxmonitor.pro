What is IRS Code 150?

Plain-English definition: IRS Code 150 means the return has been filed and assessed by the IRS, confirming the tax liability has been determined.

When does Code 150 appear?

  • 2–6 weeks after filing: Typically appears after e-filed returns are received and processed
  • With simple returns: May appear faster if return is straightforward
  • With complex returns: May take longer if additional review or schedules are needed

What does Code 150 usually signal?

If this appears alone

The return was assessed and no further action is visible yet. The refund or payment processing may be pending, or the transcript was pulled mid-cycle.

If this appears with Code 806 or 766/768

Code 150 confirmed the refund; Code 806 means it's being applied to next year's tax, while 766/768 means it's being issued as a check or direct deposit.

If this appears with Code 290 or 300

Code 150 confirmed a tax liability is due. The following code indicates a payment is owed and may be pending.

If this appears with Code 570

The refund was assessed but is temporarily held pending resolution of a discrepancy (W-2 match, ID verification, missing documentation, etc.).

What should a tax professional do next?

  1. Confirm context: Review the date, adjacent codes, and amounts to understand the assessment timeline.
  2. Validate with client: Compare transcript codes to client records (returns filed, payments made, refunds expected).
  3. Communicate clearly: Provide a short written summary and next steps (refund timeline, payment due, hold reason, etc.).

FAQ

What is IRS Code 150?

IRS Code 150 indicates that the IRS has filed and assessed the return. It's a critical milestone that signals the return has been processed and the tax liability has been determined.

Does Code 150 mean my client's refund is approved?

Not necessarily. Code 150 means the return was assessed. If there's a refund due, it will be paired with Code 806 (applied to next year) or 766/768 (issued). If Code 290 or 300 appears, a payment is due instead.
