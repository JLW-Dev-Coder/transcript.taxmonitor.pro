IRS Code 520 Meaning

IRS Code 520 is a litigation or bankruptcy freeze indicator. It shows that legal proceedings are affecting the IRS's ability to collect. This page helps you read Code 520, understand its closing codes, and explain what is—and is not—protected for your client.

What Is IRS Code 520?

IRS Code 520 typically indicates that a litigation or bankruptcy action has placed the account into a freeze status. The code does not, by itself, specify the type of case, but it tells you that normal collection activity may be restricted while the matter is pending.

You will often see Code 520 accompanied by a closing code or later entries that explain how the freeze ended and what happened next.

When Does Code 520 Appear on an IRS Transcript?

Code 520 appears when a qualifying litigation or bankruptcy event is recorded by the IRS. The transaction date will usually align with the date the case is filed or the IRS becomes aware of the proceeding.

Look at what comes before and after Code 520—especially assessments like Code 150, adjustments like Code 290, and notices under Code 971—to understand which balances are affected.

Common 520 Closing Codes and What They Mean

Different 520 closing codes (not all shown on every transcript) can indicate whether the case ended in discharge, dismissal, or some other resolution. The closing information helps you determine if balances were written down, discharged, or simply resumed for collection.

How Code 520 Affects the Collection Statute (CSED)

Litigation and bankruptcy events often toll the Collection Statute Expiration Date (CSED). While Code 520 is active, the clock may stop running on the 10-year collection period, then restart (with adjustments) when the case closes. Accurate CSED calculations require you to account for these freeze periods.

How Code 520 Interacts with Other Collection Activity

During a 520 freeze, some enforced collection actions may not proceed, but that does not always mean the IRS is hands-off. Interest may continue to accrue, and certain automated notices may still generate. Use the surrounding codes and notice entries to see what continued and what paused.

What Should a Tax Professional Do When Code 520 Appears?

  • Confirm the legal context. Determine whether the freeze is tied to bankruptcy, litigation, or another type of proceeding.
  • Identify affected periods. Map which tax years and balances fall within the 520 freeze window.
  • Recalculate CSED. Adjust collection statute estimates to reflect tolling during the freeze.
  • Plan post-closing strategy. Once the case closes, reassess options such as installment agreements, offers, or other resolutions.

FAQ: IRS Code 520

What does IRS Code 520 mean on an IRS transcript?

It signals a litigation or bankruptcy freeze that limits certain collection actions while the case is active.

Does Code 520 stop all IRS collection activity?

It often stops many enforced actions, but not necessarily all notices or interest accrual. You need to read the transcript and related legal documents to see the scope.

How do different 520 closing codes change the rules?

They indicate how the case ended, which in turn affects whether balances were discharged, reduced, or simply resume for collection.

How should I explain a 520 freeze to a client?

Present it as a legal pause that changes the collection timeline, not a permanent resolution. Then outline what planning is needed once the case closes.
