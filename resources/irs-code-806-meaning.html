IRS Code 806 Meaning

IRS Code 806 is the withholding credit code. It shows how much federal income tax withheld from W‑2s and 1099s has been applied to the account. This page explains how to read Code 806, why it often appears as a negative amount, and how it fits into refund or balance-due math.

Plain-English definition

IRS Code 806 represents the total federal income tax withholding credited to the taxpayer's account for the year. It aggregates withholding from Forms W‑2 and 1099 that has been matched and applied to the return.

When does Code 806 appear?

Code 806 typically posts around the time the return is processed and assessed under Code 150. It may appear before or after the assessment line depending on processing sequence.

What does Code 806 usually signal?

Negative vs. positive amounts

On transcripts, credits are usually displayed as negative numbers because they reduce liability. A negative Code 806 amount generally means withholding has been applied in the taxpayer’s favor.

Mismatch with the filed return

A mismatch between Code 806 and withholding reported on the return can signal missing information returns, processing errors, or identity/verification issues. Reconcile against the filed return and the wage & income transcript.

With a refund hold

When Code 806 appears with Code 570, it usually means the credit is posted, but the IRS is pausing release of any refund until additional checks complete.

What should a tax professional do next?

  • Reconcile to the return. Confirm the 806 total matches reported withholding.
  • Cross-check wage & income. Verify all payer statements are present and properly attributed.
  • Explain the bridge. Tie Code 150 (assessment) + 806 (credits) + any hold codes to the client-friendly story.

FAQ

What does IRS Code 806 mean on a tax transcript?

It records the credit for federal income tax withheld from Forms W‑2 and 1099 applied to the account.

Why does Code 806 show as a negative number?

Credit entries often display as negative amounts because they reduce the tax liability.

What if Code 806 doesn't match the withholding reported on the return?

Reconcile the return to the transcript and validate wage & income transcript data for missing or mismatched payer information.

How does Code 806 interact with refund holds like Code 570?

Code 806 contributes to the refund calculation, but Code 570 can pause the refund until the IRS completes additional checks.
