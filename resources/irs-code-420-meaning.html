IRS Code 420 Meaning

IRS Code 420 is an examination indicator. It shows that the IRS opened an exam case on the return. This page walks you through what Code 420 means, how it interacts with Codes 424 and 300, and how to translate that activity into clear client guidance.

What Is IRS Code 420?

IRS Code 420 typically carries a description like "Examination of Tax Return." It reflects that the IRS established an exam case on the account, often after an early-stage selection signal like Code 424.

For practitioners, Code 420 is your cue to ask: What issue was examined, what period was covered, and did the exam lead to any additional tax or no-change results?

When Does Code 420 Appear on an IRS Transcript?

Code 420 usually appears after the return has been assessed under Code 150 and before any deficiency assessment under Code 300. You may see it:

  • Shortly after a selection entry like Code 424.
  • Following a notice that the return was chosen for audit.
  • Months or years after filing, depending on exam workload and selection criteria.

What Does Code 420 Usually Signal?

If Code 420 Appears with Code 424

When you see Codes 424 and 420 together, 424 is the early "pulled for review" signal and 420 is the established exam. That sequence tells you the return moved from potential selection into an active case.

If Code 420 Later Pairs with Code 300

If Code 300 posts after 420, the exam resulted in a deficiency assessment. The Code 300 amount is the dollars-and-cents outcome of the exam. Your report should connect 420 → 300 so the client sees how an audit indicator turned into a bill.

If Code 420 Closes with Code 421 (No Change)

In some transcripts, you will see Code 421 (or a similar closing code) after 420, indicating a no-change or closed exam. In that scenario, there may be no Code 300 deficiency, but the transcript documents that the return was examined and closed without additional tax.

How to Read Audit Progress From the Transcript

To tell a clear audit story from the transcript:

  • Start with selection. Identify Code 424 and any related 971 notices that describe the initial contact.
  • Track exam activity. Use Code 420 dates to see when the case was opened and how long it remained active.
  • Look for outcomes. Scan for Code 300 entries or closing codes that show how the exam ended.

What Should a Tax Professional Do When Code 420 Appears?

When you encounter Code 420, you should:

  • Confirm the scope of the examination and the years or issues involved.
  • Collect all related notices, exam reports, and closing letters from the client.
  • Reconcile the transcript entries with the exam findings and any assessed deficiencies.
  • Document recommendations on appeals, payment options, or further resolution work.

FAQ: IRS Code 420

What does IRS Code 420 mean on a tax transcript?

It shows that the IRS opened an examination case on the return. It is your signal that an audit was initiated for that year.

Does Code 420 always mean a full audit?

Not necessarily. The scope can be narrow or broad, but Code 420 confirms that some level of examination activity occurred.

What's the difference between Codes 420 and 424?

Code 424 is the selection or request for examination; Code 420 indicates an established exam case.

How should I explain Code 420 to a worried client?

Use the transcript to show when the exam opened, what the outcome was, and whether any additional tax was ultimately assessed. Then discuss next steps, including appeals or payment strategies.
