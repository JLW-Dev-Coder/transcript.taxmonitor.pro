IRS Code 290 Meaning

What is IRS Code 290?

Plain-English definition: IRS Code 290 indicates that additional tax has been assessed or an internal adjustment has been made to the account. It modifies or recalculates the tax liability previously established.

When does Code 290 appear?

  • Shortly after initial assessment: To correct math errors or processing issues
  • After an examination notice: When the IRS completes an audit or exam
  • Following math-error corrections: Automated recalculations by the IRS
  • Months or years later: As part of collection or correspondence actions

What does Code 290 usually signal?

If this shows a positive amount

The IRS has increased the assessed tax. This can reduce or eliminate a refund, create a new balance due, or increase an existing balance owed. Walk the client from the original assessment (Code 150) through the 290 amount to show where the change impacts the bottom line.

If this shows $0.00

A $0 Code 290 often reflects an internal adjustment or closing action where the IRS documented activity but did not change the net tax. The related notice may explain a review, reclassification, or system correction without affecting the final balance.

If this appears with Code 150 and Code 806

You're usually looking at a complete reconciliation sequence. Code 150 sets the original tax, Code 290 adjusts it, and Code 806 applies withholding or credits. Model this as a bridge: "Original tax → adjustment → credits → final balance/refund."

If this appears with Code 971 (Notice Issued)

Code 290 and Code 971 together typically mark a formal notice about the change. The 971 line may reference a CP or letter number that you can match to the physical notice. This is your evidence trail: "Here is when the IRS changed the account, and here is the letter they sent."

If this appears with Code 300 (Examination)

Code 290 may follow an exam and represent the deficiency or proposed adjustment from the examination. Check the date sequence and any related 971 entries to understand what the IRS examined and what they changed.

How Code 290 affects balances and refunds

Code 290 is an assessment-side entry. Pair it with payments and credits to show whether the account moves toward a refund, zero balance, or balance due. The key is context: the amount posted, the date, and nearby notice/activity codes.

What should a tax professional do next?

  1. Identify the trigger: Use surrounding codes, especially Code 971, to find the notice or event that caused the adjustment.
  2. Rebuild the math: Recreate the tax calculation before and after Code 290 so the client sees what changed.
  3. Check statute dates: Note whether the timing affects assessment or collection limitation periods.
  4. Communicate clearly: Provide a written summary in a transcript report, not just internal notes.

Does Code 290 always mean more tax is owed?

No. A positive 290 increases tax owed, but a $0 entry may document an internal adjustment with no net change. Always check the amount and nearby codes to understand the full impact.
