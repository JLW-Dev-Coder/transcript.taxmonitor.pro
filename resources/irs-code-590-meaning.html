IRS Code 590 Meaning

IRS Code 590 flags that the IRS closed a non-filer inquiry for a tax year as no return filed. On an account transcript, it tells you the IRS did not have a return on file as of that date and has moved the case into a non-filer resolution status. What is IRS Code 590?

Plain-English definition: Code 590 is a non-filer closing code that tells you the IRS has decided a return is not on file for the module. It often appears when the IRS reviews a potential filing requirement and closes the case as "no return secured" or determines no return is required.

When does Code 590 appear?

  • After a non-filer inquiry: The IRS may post 590 when it has contacted the taxpayer about a missing return and closed the case without securing one.
  • When no filing requirement exists: In some cases, 590 documents that the IRS concluded the taxpayer was not required to file for that period.
  • Before or alongside assessment codes: In higher-risk situations, 590 can appear in the timeline before a substitute-for-return or other assessment sequence posts.

What does Code 590 usually signal?

If the client truly had no filing requirement

Code 590 can support your conclusion that the IRS evaluated the year and closed it with no return required. You still need workpapers backing how you reached that conclusion under income and filing threshold rules.

If the client should have filed but did not

Here, 590 is a warning sign. It shows the IRS recognized a potential non-filer issue and closed it without receiving a return. That can precede compliance actions, substitute-for-return assessments, or additional notices.

What should a tax professional do next?

  1. Confirm filing requirement: Use wage and income transcripts, prior year returns, and thresholds to determine whether a return was required for the year in question.
  2. Scan for assessments: Look for Codes 150, 290, or 300 that show whether the IRS has already assessed a liability without a filed return.
  3. Document a clear plan: In your report, spell out whether a return should be filed now, whether an existing substitute-for-return must be corrected, or whether the year can be documented as not required.

FAQ

What does IRS Code 590 mean?

IRS Code 590 means the IRS has closed a non-filer case for that tax period with a conclusion of no return filed. It tells you that, on the posting date, the IRS did not have a signed return on file and resolved the non-filer inquiry under its closing code procedures.

Does Code 590 trigger substitute-for-return action?

Not by itself. Code 590 documents the non-filer case outcome. To evaluate substitute-for-return exposure, review the transcript for assessment codes like 150 or 290 that show the IRS calculated a liability without a taxpayer-filed return.

What should I verify before advising a client?

Verify whether a filing requirement existed, check for posted assessments, compare dates to client documents, and review wage and income data. Your workpapers should explain clearly whether a late-filed return is needed or whether the year can remain closed.

Can Code 590 appear after a return was filed?

Yes. You may see Code 590 from an early non-filer action, followed later by Codes 594, 597, or 599 and Code 150 when a return eventually posts. Always read the full sequence in date order before telling a client what the IRS believes about that year.
