IRS Code 971 Meaning

IRS Code 971 is the transcript's "notice issued" flag. Each 971 entry points to an IRS letter or miscellaneous action. This page shows you how to pair 971 with other codes so you can explain what the IRS said and why, without guesswork.

Plain-English definition

IRS Code 971 marks that the IRS issued a notice or took a miscellaneous action related to the account. It does not, by itself, tell you the letter number, but it anchors the timing of IRS communications in the transcript.

When does Code 971 appear?

Code 971 can appear at many points in the life of a case:

  • After adjustments such as Code 290 (Additional Tax Assessed).
  • Alongside refund holds like Code 570.
  • Near exam codes such as Code 420 and Code 424.

The key is to read 971 in context: what changed on the account just before the notice issued, and what happened afterward?

What does Code 971 usually signal?

Adjustment or correction notice

When Code 971 appears near an adjustment like Code 290, it often represents the notice explaining the change. Use the pairing to connect the transcript movement to the client's mail.

Refund hold or verification notice

If Code 971 is paired with a hold under Code 570, the notice typically explains why the IRS paused a refund and what the taxpayer needs to do (if anything).

Audit or review correspondence

In exam contexts, 971 entries can log initial contact letters, information requests, or closing letters associated with Code 420 and Code 424.

What should a tax professional do next?

  • Build a notice timeline. List each 971 date, the nearby codes, and the matching letter/CP notice.
  • Confirm client receipt. Document whether the client received and responded to each notice.
  • Check deadlines. Identify response windows that may be running from the notice date.
Example transcript sequence
971  Notice issued (client letter/CP)
290  Additional tax assessed or adjustment
570  Additional account action pending (hold)

FAQ

What does IRS Code 971 mean on a transcript?

It indicates that the IRS issued a notice or took a miscellaneous action recorded for that date.

How do I know which notice matches a 971 entry?

Match the 971 date and any brief description to the letters the client received, then use the surrounding codes to infer the reason.

What should I do when Code 971 appears with other codes?

Treat 971 as the communication marker. Use the nearby codes (like 290 or 570) to explain what changed and what the IRS is waiting on.

Can multiple Code 971 entries appear in one year?

Yes. Each 971 entry is a separate notice or action. Build a timeline so the client sees the sequence clearly.
